StratusLab wins runner-up prize in poster competition at the EGI Technical Forum

Thanks to all who attended the StratusLab sessions at the EGI Technical Forum in Lyon last week. David O'Callaghan chaired an interesting session on Virtualisation and Cloud Computing while Marc-Elian Begin led the session on Using Agile in Academic, national and European projects.

The StratusLab tutorial on Thursday had users creating VMs on a StratusLab cloud and using the StratusLab Martketplace to find and share virtual machine images.

And finally Christophe Blanchet's poster on Bioinformatics appliances won the runner-up prize in the poster competition! Christophe's poster described the two bioinformatics virtual appliances that have been created by his team at the Institut de Biologie et Chimie des Protéines CNRS: a Biological databases repository and a Bioinformatics compute node. These virtual appliances, which can be quickly deployed on any StratusLab cloud, contain the software and data needed to run complex genome sequencing. Scientists with a cloud login can launch multiple instances of the virtual appliances in order to run their analyses more quickly than would be possible using local resources.
