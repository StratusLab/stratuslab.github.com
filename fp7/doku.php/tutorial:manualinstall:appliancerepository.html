Appliance Repository Manual Installation

To install your own instance of an Appliance Repository, simply configure the corresponding parameters and run the install command with the –apprepo option.

Authentication

The Appliance Repository can be configure to authenticate based on a local Apache/HTTPD file or using an existing LDAP server. Here is how to configure StratusLab for both scenarios:

LDAP

Here is an extract from the app-repo section of the stratuslab.cfg.ref file of the parameters that must be configured to use LDAP for authentication: 

# Appliance Repository
app_repo_url = https://appliances.stratuslab.eu/images

# Directory in which to store the images
app_repo_image_dir = /images

# Structure of the repository
app_repo_structure = #type_#/#os#-#osversion#-#arch#-#type#/#version#

# Repository image filename structure
app_repo_filename = #os#-#osversion#-#arch#-#type#-#version#.img.#compression#

# URL of the LDAP server to use for authentication
app_repo_ldap_url = ldap://ldap.stratuslab.eu:10636/ou=people,o=stratuslab?uid'

# Enable LDAP authentication
app_repo_use_ldap = True

# Certificate to use for authenticating the LDAP server
app_repo_ldap_cert = 

# LDAP Bind DN
app_repo_ldap_bind = cn=repo,ou=daemons,o=stratuslab

# Password to use when connecting to the LDAP server
app_repo_ldap_passwd = 

# Location of Apache configuration
app_repo_apache_home = /etc/httpd

Apache2/HTTPD Authentication

Here is an extract from the app-repo section of the stratuslab.cfg.ref file of the parameters that must be configured to use local Apache2/HTTPD authentication: 

# Appliance Repository
app_repo_url = https://appliances.stratuslab.eu/images

# Directory in which to store the images
app_repo_image_dir = /images

# Structure of the repository
app_repo_structure = #type_#/#os#-#osversion#-#arch#-#type#/#version#

# Repository image filename structure
app_repo_filename = #os#-#osversion#-#arch#-#type#-#version#.img.#compression#

# URL of the LDAP server to use for authentication
app_repo_ldap_url = ldap://ldap.stratuslab.eu:10636/ou=people,o=stratuslab?uid'

# Enable LDAP authentication
app_repo_use_ldap = False

# Location of Apache configuration
app_repo_apache_home = /etc/httpd

# HTTPD user password file (required if not using LDAP)
app_repo_httpd_passwd_file =

Installation

Once the configuration is complete, simply run the install command: 

stratus-install --apprepo

Testing the Installation

To test the Appliance Repository installation, you can use the stratus-upload-image command, which is part of the stratuslab-cli-user package: 

$ stratus-upload-image ...
