Appliance Repository

The appliance repository is a service provided by StratusLab to encourage the sharing of machine images. The repository contains both base images and appliances.

Base Images

Base images are minimal (or small) installations of standard operating systems. They are intended mainly as a starting point for creating fuller, customized images, but are functional in their own right. Typical base images use operating systems like CentOS 5.5, Ubuntu 10.04, and ttylinux 9.4.

Appliances

Appliances are customized images designed to run a single service. They typically contain a minimal OS with just the dependencies required for the provided service. Eventually, these images will be provided by members of the community as well as by StratusLab.

Browsing the Repository

The appliance repository is based on standard web technologies: Apache Web Server with WEBDAV support. You can point your web browser to the repository to see what images are available.

Find the ttylinux-9.5 base image that we will be using throughout this tutorial. The actual image has a *.img.gz extension; there is also an associated xml file containing metadata about the image. You might want to download the metadata to see what information it contains.
