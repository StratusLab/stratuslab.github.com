StratusLab Release v0.1

StratusLab releases first open source cloud solution designed for the grid.

The StratusLab project has released the first version of its cloud computing distribution which aims to provide a full cloud solution for grid and cluster computing.

The release is a technology preview and not yet production-ready, but it will give system administrators and users a chance to try out the new features of what will become an integrated solution for cloud management and running grid services within the cloud.

This StratusLab release contains the following:

  • OpenNebula: virtual machine management
  • Web Monitor: administrator dashboard for cloud
  • User CLI: Python-based command line tools for remote access to cloud
  • System Admin. CLI: Command line tools to facilitate installation and management
  • Manual and Automated (Quattor) Installation

Installation

The reference deployment (see below) of the StratusLab distribution can be installed via a manual installation (simplified with StratusLab scripts) or via Quattor. See the appropriate documentation for the method you would like to use.

Support

Access to the Cloud Infrastructure

The project maintains a cloud infrastructure for tests. We will allow access to a limited number of people external to the project. If you are interested in testing the StratusLab distribution, please send an email to the support mailing list.

Packages

The software packages for this release are:

  • OpenNebula (one-2.0-StratusLab-0.1.0-1, RPM)
  • Command Line Interface for Users (stratuslab-cli-user-1.1-1, TAR, RPM)
  • Command Line Interface for Sys. Admins (stratuslab-cli-sysadmin-1.1-1, TAR, RPM)
  • StratusLab Web Monitor (stratuslab-web-monitor-1.1-1, RPM)
  • Benchmarks (stratuslab-benchmarks-1.0-1, RPM)
  • OpenNebula Quattor Config. Module (ncm-oned-1.2.1-1, RPM)
  • Libvirtd Quattor Config. Module (ncm-libvirtd-1.0.1-1, RPM)
  • Quattor Cloud Config. (quattor-cloud-config-0.1.1, TAR)
  • New in v0.2: Cloud Proxy (stratuslab-cloud-proxy, RPM)

The release also includes virtual machine base images:

  • ttylinux 9.5
  • Ubuntu 10.04
  • CentOS 5.5

which are available from the appliance repository.
