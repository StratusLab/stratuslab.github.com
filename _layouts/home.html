What's Behind

StratusLab provides features for dynamic management of
  • Compute - thin provisioning, fast boot-up
  • Storage - EBS-like persistent writes, snapshots, etc
  • Network resources - public and private networks
We think that's expected. But further, it also provides:
  • Uniform (but flexible) authentication and authorization system
  • Shared image marketplace across several cloud instances
  • Tools for simplifying image creation
  • Facilities for federating StratusLab cloud infrastructures
... and all of this, out-of-the-box!

Resource centers deploying the software benefit from improved and simplified resource management; users of those resource centers benefit from a more dynamic and customized computing environment, simplifying the execution of their scientific and commercial applications.
