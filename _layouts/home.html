Darn simple cloud

StratusLab provides an Infrastructure as a Service (IaaS) cloud distribution that is both easy to install and use.

Like other cloud distributions it provides services for the dynamic management of compute, storage, and network resources. Unlike other cloud distributions, it provides:

  • Uniform (but flexible) authentication and authorization system,
  • Services for virtual machine image management, and
  • Facilities for the federation of different cloud infrastructures
      for a complete, functional cloud infrastructure.

      Resource centers deploying the software benefit from improved and simplified resource management; users of those resource centers benefit from a more dynamic and customized computing environment, simplifying the execution of their scientific and commercial applications.
